## 🎯 下一步操作（需要您手动完成）

### 快速版（5分钟）

1. **访问Pixabay免费音乐库**：
   ```
   https://pixabay.com/music/search/chinese/
   ```

2. **选择并下载一首中国风音乐**：
   - 推荐搜索关键词：`chinese traditional`, `guzheng`, `meditation asian`
   - 点击任一曲目的"Free Download"按钮
   - 下载MP3格式

3. **将文件移到项目BGM目录**：
   ```bash
   mv ~/Downloads/你下载的文件.mp3 storyboards/文脉薪传/bgm/traditional_chinese.mp3
   ```

4. **运行BGM添加命令**：
   ```bash
   source venv/bin/activate

   python add_bgm_to_video.py \
     --video "storyboards/文脉薪传/文脉薪传_完整版.mp4" \
     --bgm "storyboards/文脉薪传/bgm/traditional_chinese.mp3" \
     --output "storyboards/文脉薪传/文脉薪传_最终版.mp4" \
     --volume 0.12
   ```

5. **完成！** 您将得到最终版视频：
   ```
   storyboards/文脉薪传/文脉薪传_最终版.mp4
   ```

## 🔧 BGM音量调整参考

| 场景 | 推荐音量 | 说明 |
|------|---------|------|
| 纯背景衬托 | 0.08-0.10 | 非常轻柔，不抢画外音 |
| **标准配置** | **0.12-0.15** | **推荐，平衡度最佳** |
| 突出音乐氛围 | 0.18-0.20 | 较响亮，适合音乐性强的场景 |

**调整方法**：
```bash
# 如果音量不合适，修改 --volume 参数重新运行
python add_bgm_to_video.py \
  --video "storyboards/文脉薪传/文脉薪传_完整版.mp4" \
  --bgm "storyboards/文脉薪传/bgm/traditional_chinese.mp3" \
  --output "storyboards/文脉薪传/文脉薪传_最终版.mp4" \
  --volume 0.15  # 修改这里的值
```
