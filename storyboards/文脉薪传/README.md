## 📖 项目简介

本项目是一个以"文脉薪传"为主题，聚焦儒家"仁义礼智信"五常传承的文化主题短视频的**完整分镜设计**。

包含从创意策划到分镜可视化的全部前期工作，可直接用于视频制作。

### 🎯 核心特色

- ✅ **13个详细场景**，每个场景包含完整的设计说明
- ✅ **AI图像生成提示词**，可直接用于Midjourney/DALL-E
- ✅ **分镜可视化图**，包含网格图、时间轴、单帧图
- ✅ **自动化工具**，可快速修改和重新生成
- ✅ **完整文档**，从制作指南到技术规格

## 💡 使用AI生成图片

每个场景都包含专业的英文提示词，可直接用于：

### Midjourney
```
复制场景的 image_generation_prompt
在Discord中输入：/imagine prompt: [提示词]
选择最佳结果并放大
```

### DALL-E 3
```
在ChatGPT Plus中
直接粘贴提示词
下载生成的图片
```

### Stable Diffusion
```
使用提示词 + 参数调整
本地生成或在线服务
批量生成多个版本
```
