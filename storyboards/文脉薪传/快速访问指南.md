## 🔄 如何修改

### 修改脚本内容
```bash
# 1. 编辑脚本文件
code 文脉薪传_细化脚本.yaml

# 2. 修改场景、旁白、时长等

# 3. 重新生成设计文档
python generate_storyboard_images.py \
  --script 文脉薪传_细化脚本.yaml \
  --output ./storyboards/文脉薪传

# 4. 重新生成可视化
python create_storyboard_visualization.py

# 5. 查看更新后的文件
open storyboards/文脉薪传/visualizations/storyboard_grid.png
```

### 调整时长
```yaml
# 在脚本中找到场景，修改 duration 字段
scenes:
  - id: scene_01_opening
    duration: 20  # 从15秒改为20秒
    narration: "..."
```

### 修改色彩方案
```yaml
# 找到 color_palette 部分
color_palette:
  primary: '#2C1810'    # 修改主色调
  secondary: '#D4A574'  # 修改辅助色
  accent: '#8B0000'     # 修改强调色
```

## 🆘 常见问题

### Q1: 找不到某个文件？
```bash
# 使用 find 命令搜索

find . -name "*.png"
```

### Q2: 图片打不开？
```bash
# 检查文件是否存在
ls -lh storyboards/文脉薪传/visualizations/storyboard_grid.png

# 重新生成
python create_storyboard_visualization.py
```

### Q3: 需要导出为PDF？
```bash
# 使用 pandoc 转换 Markdown 为 PDF
pandoc storyboards/文脉薪传/分镜设计文档.md -o 分镜设计文档.pdf

# 或使用在线工具
# https://www.markdowntopdf.com/
```

```bash
code create_storyboard_visualization.py

```
