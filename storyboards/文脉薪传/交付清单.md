## 🛠️ 工具脚本（可复用）

### 1. 分镜图像生成脚本
**文件**：`generate_storyboard_images.py`
- **功能**：
  - 解析YAML脚本
  - 提取图像生成提示词
  - 生成设计文档（JSON + Markdown）
  - （可选）使用Claude AI生成设计建议
- **用法**：
  ```bash
  python generate_storyboard_images.py \
    --script 文脉薪传_细化脚本.yaml \
    --output ./storyboards/文脉薪传
  ```

### 2. 分镜可视化脚本
**文件**：`create_storyboard_visualization.py`
- **功能**：
  - 生成分镜网格图
  - 生成单独分镜框
  - 生成时间轴视图
- **用法**：
  ```bash
  python create_storyboard_visualization.py \
    --design ./storyboards/文脉薪传/complete_storyboard_design.json \
    --output-dir ./storyboards/文脉薪传/visualizations
  ```
