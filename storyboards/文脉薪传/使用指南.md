## 🎯 下一步操作指南

### 方案一：使用 AI 图像生成工具创建分镜参考图

#### 1. 准备图像生成提示词
每个场景的设计文件中都包含 `image_generation_prompt` 字段，可以直接用于：
- **Midjourney**: 复制提示词到 Discord
- **DALL-E 3**: 通过 ChatGPT Plus 或 API
- **Stable Diffusion**: 本地部署或在线服务

#### 2. 生成分镜参考图
```bash
# 提取所有提示词到文本文件
python -c "
import json
with open('storyboards/文脉薪传/complete_storyboard_design.json', 'r') as f:
    data = json.load(f)
    for scene in data['scenes']:
        print(f\"\\n=== {scene['scene_id']} ===\")
        print(scene['original_prompt'])
" > prompts.txt
```

#### 3. 保存生成的图片
将生成的图片保存为：
- `scene_01_opening_design.png`
- `scene_02_ren_intro_design.png`
- ... 依此类推

### 方案二：直接制作视频（无需生成参考图）

由于这是一个**纯文化阐述类视频**，不涉及产品演示或网站操作，我们需要使用不同的方法：

#### 选项 A：使用 Python 动画库创建视频

创建一个新的脚本来生成动画视频，包含：
- 文字动画
- 图像素材（需要准备）
- 水墨效果
- 转场动画

#### 选项 B：使用专业视频制作工具

1. **After Effects**: 最适合水墨风格和中国风动画
2. **Premiere Pro**: 适合剪辑和组合素材
3. **DaVinci Resolve**: 免费且功能强大
4. **剪映**: 国产软件，有丰富的中国风模板

## 📝 创意执行建议

### 视觉风格实现

#### 1. 水墨动画效果
```
- 使用水墨笔刷素材
- 逐帧动画展现笔触
- 黑白为主，点缀朱砂红和青花蓝
- 参考软件：After Effects + 水墨插件
```

#### 2. 文字动画
```
- 书法字体：方正清刻本悦宋
- 出现方式：笔画逐渐书写
- 配合毛笔音效
- 工具：AE 的 Stroke 效果 + Trim Paths
```

#### 3. 转场效果
```
- 主转场：水墨晕染（使用 Turbulent Displace）
- 次转场：交叉溶解
- 特殊转场：毛笔挥洒（使用粒子系统）
```

### 音频制作

#### 1. 配音生成
```bash
# 使用项目自带的 TTS 引擎生成配音
source venv/bin/activate
python -m demo_video_generator.cli.main audio \
  --script 文脉薪传_细化脚本.yaml \
  --output ./output/audio/文脉薪传 \
  --voice zh-CN-YunxiNeural
```

#### 2. 背景音乐建议
- **开篇**: 古琴曲《流水》或《梅花三弄》
- **仁**: 柔和弦乐，如《春江花月夜》
- **义**: 激昂鼓点，如《十面埋伏》
- **礼**: 编钟音乐，如曾侯乙编钟演奏
- **智**: 现代电子+古典融合
- **信**: 稳重低音提琴
- **结尾**: 交响乐《红旗颂》或《梁祝》
